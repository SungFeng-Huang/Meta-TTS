# Preprocess

```
python preprocess_v2.py [raw_dir] [preprocessed_dir] --dataset [DATASET_TAG] [--parse_raw] [--prepare_mfa] [--mfa] [--preprocess] [--create_dataset] [--force]
```

```DATASET_TAG``` can be one of ```LibriTTS```, ```AISHELL-3```, ```KSS```, ```JSUT```, ```CSS10```, ```TAT```, ```TAT_TTS```
